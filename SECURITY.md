# Security Policy

## Supported Versions

We support security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| > 9.0   | :white_check_mark: |
| 8.9.0   | :white_check_mark: |
| < 8.9.0 | :x:                |

## Reporting a Vulnerability

To report a vulnerability please write a message to `io@gpbl.dev`.
