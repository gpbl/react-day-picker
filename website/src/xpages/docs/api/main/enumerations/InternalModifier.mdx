# InternalModifier

The name of the modifiers that are used internally by DayPicker.

## Enumeration Members

| Enumeration Member | Value | Description |
| :------ | :------ | :------ |
| <a id="disabled" name="disabled"></a> `Disabled` | `"disabled"` | Name of the modifier applied to the disabled days, using the `disabled` prop. |
| <a id="hidden" name="hidden"></a> `Hidden` | `"hidden"` | Name of the modifier applied to the hidden days using the `hidden` prop). |
| <a id="outside" name="outside"></a> `Outside` | `"outside"` | - |
| <a id="rangeend" name="rangeend"></a> `RangeEnd` | `"range_end"` | The modifier applied to the day ending a selected range, when in range selection mode. |
| <a id="rangemiddle" name="rangemiddle"></a> `RangeMiddle` | `"range_middle"` | The modifier applied to the days between the start and the end of a selected range, when in range selection mode. |
| <a id="rangestart" name="rangestart"></a> `RangeStart` | `"range_start"` | The modifier applied to the day starting a selected range, when in range selection mode. |
| <a id="selected" name="selected"></a> `Selected` | `"selected"` | Name of the modifier applied to the selected days using the `selected` prop). |
| <a id="today" name="today"></a> `Today` | `"today"` | Name of the modifier applied to the day specified using the `today` prop). |
