# Container Attributes

Pass selected HTML attributes and `data-` attributes to the root element.

<ContainerAttributes />

```tsx
export function ContainerAttributes() {
  return (
    <DayPicker
      id="testId"
      className="testClass"
      data-test="testData"
      nonce="foo_nonce"
      title="foo_title"
      lang="it"
      mode="multi"
    />
  );
}
```
