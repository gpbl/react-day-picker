# useDayRender()

<a id="undefined" name="undefined"></a>

> **useDayRender**(`day`, `displayMonth`, `buttonRef`): [`DayRender`](/docs/api/main/type-aliases/DayRender.md)

Return props and data used to render the [Day](/docs/api/main/functions/Day.md) component.

Use this hook when creating a component to replace the built-in `Day`
component.

## Parameters

| Parameter | Type | Description |
| :------ | :------ | :------ |
| `day` | `Date` | The date to render. |
| `displayMonth` | `Date` | The month where the date is displayed (if not the same as `date`, it means it is an "outside" day). |
| `buttonRef` | `RefObject`\<`HTMLButtonElement`\> | A ref to the button element that will be target of focus when rendered (if required). |

## Returns

[`DayRender`](/docs/api/main/type-aliases/DayRender.md)
