title: Matching Days
---

The `selected`, `disabled`, `hidden` props, and any [modifier](./modifiers.mdx), accept different types useful to match days.

## Match a Single Day

Pass a `Date` object to match a single day.

```jsx preview
import React, { useState } from 'react';
import { DayPicker } from 'react-day-picker';

function App() {
  const [month, setMonth] = useState(new Date(2019, 8));
  return (
    <DayPicker
      month={month}
      onMonthChange={setMonth}
      selected={new Date(2019, 8, 9)}
    />
  );
}
```

## Match Multiple Days

Pass an array of `Date` objects to match multiple days.

```jsx preview
import React, { setState } from 'react';
import { DayPicker } from 'react-day-picker';

function App() {
  const [month, setMonth] = useState(new Date(2019, 8));
  return (
    <DayPicker
      month={month}
      onMonthChange={setMonth}
      selected={[
        new Date(2019, 8, 9),
        new Date(2019, 8, 10),
        new Date(2019, 8, 11),
      ]}
    />
  );
}
```

## Match Days between a Range

Pass a `{ before, after }` object.

```jsx preview
import React, { setState } from 'react';
import { DayPicker } from 'react-day-picker';

function App() {
  const [month, setMonth] = useState(new Date(2019, 8));
  return (
    <DayPicker
      month={month}
      onMonthChange={setMonth}
      selected={{
        after: new Date(2019, 8, 9),
        before: new Date(2019, 8, 19),
      }}
    />
  );
}
```

It is possible to omit the `before` or `after` keys.

```jsx preview
import React, { setState } from 'react';
import { DayPicker } from 'react-day-picker';

function App() {
  const [month, setMonth] = useState(new Date(2019, 8));
  return (
    <DayPicker
      month={month}
      onMonthChange={setMonth}
      selected={{
        after: new Date(2019, 8, 9),
      }}
    />
  );
}
```

## Match Days of the Week

Pass an array of days of the week (`0` being Sunday) to a `daysOfWeek` object.

For example, to disable Sundays and Tuesdays:

```jsx preview
import React, { setState } from 'react';
import { DayPicker } from 'react-day-picker';

function App() {
  const [month, setMonth] = useState(new Date(2019, 8));
  return (
    <DayPicker
      month={month}
      onMonthChange={setMonth}
      selected={{ daysOfWeek: [0, 2] }}
    />
  );
}
```

## Matching Anything

Pass a function to have complete control. The function receive the day as argument:

```jsx preview
import React, { setState } from 'react';
import { DayPicker } from 'react-day-picker';

function App() {
  const [month, setMonth] = useState(new Date(2019, 8));
  const isBirthday = day => day.getDate() === 19 && day.getMonth() === 8;

  return (
    <DayPicker month={month} onMonthChange={setMonth} selected={isBirthday} />
  );
}
```

## A Combination of All

Pass an array to mix multiple types:

```jsx preview
import React, { setState } from 'react';
import { DayPicker } from 'react-day-picker';

function App() {
  const [month, setMonth] = useState(new Date(2019, 8));
  const isBirthday = day => day.getDate() === 19 && day.getMonth() === 8;

  return (
    <DayPicker
      month={month}
      numberOfMonths={2}
      selected={[
        isBirthday,
        {
          after: new Date(2019, 8, 1),
          before: new Date(2019, 8, 6),
        },
        {
          after: new Date(2019, 9, 14),
          before: new Date(2019, 9, 19),
        },
        new Date(2019, 8, 21),
      ]}
      onMonthChange={setMonth}
    />
  );
}
```
