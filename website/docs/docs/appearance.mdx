# Customization Props

Adjust the calendarâ€™s look and layout with these appearance props.

| Prop Name           | Type                                                                         | Description                                                                                                                                                                                                                     |
| ------------------- | ---------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `captionLayout`     | `"label"`<br/> `"dropdown"`<br/> `"dropdown-months"`<br/> `"dropdown-years"` | Choose the layout of the month caption. Default is `label`. See [Caption and Navigation Layouts](./caption-and-nav-layouts.mdx#caption-layouts).                                                                                |
| `reverseYears`      | `boolean`                                                                    | Reverse the order of the years in the dropdown. See [Caption and Navigation Layouts](./caption-and-nav-layouts.mdx#caption-dropdown).                                                                                           |
| `navLayout`         | `"around"` \| `"after"`                                                      | Adjust the positioning of the navigation buttons. Default is `"after"`. See [Caption and Navigation Layouts](./caption-and-nav-layouts.mdx#navigation-layouts).                                                                 |
| `disableNavigation` | `boolean`                                                                    | Disable navigating between months. See [Caption and Navigation Layouts](./caption-and-nav-layouts.mdx#disablenavigation).                                                                                                       |
| `hideNavigation`    | `boolean`                                                                    | Hide the navigation bar. See [Caption and Navigation Layouts](./caption-and-nav-layouts.mdx#hidenavigation).                                                                                                                    |
| `animate`           | `boolean`                                                                    | Animate month transitions. See [Months Navigation](./navigation.mdx#animate).                                                                                                                                                   |
| `fixedWeeks`        | `boolean`                                                                    | Display 6 weeks per month. See [Grid and Months](./grid-and-months.mdx#fixed-weeks).                                                                                                                                            |
| `footer`            | `ReactNode` \| `string`                                                      | Add a footer to the calendar, acting as a [live region](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/ARIA_Live_Regions). See [Week Numbers and Footer](./week-numbers-and-footer.mdx#footer-as-live-region). |
| `hideWeekdays`      | `boolean`                                                                    | Hide the row displaying the weekday names. See [Grid and Months](./grid-and-months.mdx).                                                                                                                                        |
| `numberOfMonths`    | `number`                                                                     | The number of displayed months. Default is `1`. See [Grid and Months](./grid-and-months.mdx#multiplemonths).                                                                                                                    |
| `reverseMonths`     | `boolean`                                                                    | When displaying multiple months, reverse the order of the months. See [Grid and Months](./grid-and-months.mdx#multiplemonths).                                                                                                  |
| `pagedNavigation`   | `boolean`                                                                    | Enable paginated navigation when displaying multiple months. See [Grid and Months](./grid-and-months.mdx#multiplemonths).                                                                                                       |
| `showOutsideDays`   | `boolean`                                                                    | Display the days falling into other months. See [Grid and Months](./grid-and-months.mdx#outside-days).                                                                                                                          |
| `showWeekNumber`    | `boolean`                                                                    | Display the column with the [week numbers](./week-numbers-and-footer.mdx#showweeknumber).                                                                                                                                     |
