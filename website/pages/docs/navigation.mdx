# Months Navigation

<Description>Configure how users can navigate between months.</Description>

---

Configure how users can navigate between months.

## Change the Default Month

DayPicker displays the month of the current date. To change the default month, set the `defaultMonth` prop.

<PreviewBox preview={<Examples.DefaultMonth />}>
  ```tsx
  <DayPicker defaultMonth={new Date(1979, 8)} />
  ```
</PreviewBox>

## Control the Current Month

DayPicker controls the displayed month and stores it in its internal state. To control the current month, set it in your componentâ€™s state.

To control the current month, use `month` (as opposed to `defaultMonth`) and `onMonthChange` to handle the current month.

<PreviewBox preview={<Examples.Controlled />}>
  
  ```tsx
  const [month, setMonth] = useState(new Date());
  // ...
  <DayPicker month={month} onMonthChange={setMonth} />
  ```

</PreviewBox>

## Limit the Month Navigation

As default, DayPicker can navigate indefinitely in the past or in the future. To limit the navigable months between two years, use `fromYear` or `toYear`. You can also limit the navigation with `fromDate`/`toDate` and `fromMonth` and `toMonth`.

<PreviewBox preview={<Examples.FromToYear />}>
  ```tsx
  <DayPicker defaultMonth={new Date(2024, 1)} fromYear={2024} toYear={2026} />
  ```
</PreviewBox>

## Dropdowns Navigation

Set the `dropdownNavigation` to display a combobox to change the month and/or the year.

<PreviewBox preview={<Examples.Dropdown />}>
  ```tsx
  <DayPicker dropdownNavigation />
  ```
</PreviewBox>

## Disable or hide the Navigation

To disable the navigation between months, use `disableNavigation`.

<PreviewBox preview={<Examples.DisableNavigation />}>
  ```tsx
  <DayPicker disableNavigation />
  ```
</PreviewBox>

Hide the navigation buttons with the `hideNavigation` prop. This will not disable the navigation.

<PreviewBox preview={<Examples.HideNavigation />}>
  ```tsx
  <DayPicker hideNavigation />
  ```
</PreviewBox>

## Multiple Months

Use `numberOfMonths` to render more than one month in the calendar.

<PreviewBox preview={<Examples.MultipleMonths />}>
  ```tsx
  <DayPicker numberOfMonths={2} />
  ```
</PreviewBox>

### Paged Navigation

When rendering multiple months, toggle `pagedNavigation` to navigate the given number of months per time.

<PreviewBox preview={<Examples.MultipleMonthsPaged />}>
  ```tsx
  <DayPicker numberOfMonths={2} pagedNavigation />
  ```
</PreviewBox>
