# Type alias: Mode

> **Mode**: `"none"` \| `"single"` \| `"multi"` \| `"range"`

The available selection modes.

Available modes:

- `none`: no selection is allowed.
- `single`: only one day can be selected.
- `multi`: multiple days can be selected.
- `range`: a range of days can be selected.
