# Interface: SelectSingleContextValue

Represents the value of a [SelectSingleContext](/docs/api/main/variables/SelectSingleContext.mdx).

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `onDayClick?` | [`DayClickEventHandler`](/docs/api/main/type-aliases/DayClickEventHandler.mdx) | Event handler to attach to the day button to enable the single select. |
| `selected` | `undefined` \| `Date` | The day that has been selected. |
