# Function: useNavigation()

> **useNavigation**(): [`NavigationContextValue`](/docs/api/main/interfaces/NavigationContextValue.mdx)

Hook to access the [NavigationContextValue](/docs/api/main/interfaces/NavigationContextValue.mdx). Use this hook to navigate
between months or years in DayPicker.

This hook is meant to be used inside internal or custom components.

[`NavigationContextValue`](/docs/api/main/interfaces/NavigationContextValue.mdx)
