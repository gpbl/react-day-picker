Use the `components` prop to<br />replace the internal components, like the navigation icons.<br /><br />### Styling<br /><br />DayPicker comes with a default, basic style in `react-day-picker/style` – use<br />it as template for your own style.<br /><br />If you are using CSS modules, pass the imported styles object the<br />`classNames` props.<br /><br />You can also style the elements via inline styles using the `styles` prop.<br /><br />### Form fields<br /><br />If you need to bind the date picker to a form field, you can use the<br />`useInput` hooks for a basic behavior.
