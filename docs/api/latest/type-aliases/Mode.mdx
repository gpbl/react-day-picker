# Type alias: Mode

> **Mode**: `"single"` \| `"multiple"` \| `"range"` \| `"default"`

Selection modes supported by DayPicker.

- `single`: use DayPicker to select single days.
- `multiple`: allow selecting multiple days.
- `range`: use DayPicker to select a range of days.
- `default`: disable the built-in selection behavior. Customize what is
  selected by using [PropsBase.onDayClick](/api/latest/interfaces/PropsBase.mdx).
