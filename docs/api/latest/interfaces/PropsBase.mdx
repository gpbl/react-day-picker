# Interface: PropsBase

The base props for the [DayPicker](/api/latest/functions/DayPicker.mdx) component.

These props are used to change the navigation, the styling and the behavior
of the calendar.

## Extended by

- [`DayPickerContextValue`](/api/latest/interfaces/DayPickerContextValue.mdx)
- [`PropsDefault`](/api/latest/interfaces/PropsDefault.mdx)
- [`PropsMulti`](/api/latest/interfaces/PropsMulti.mdx)
- [`PropsRange`](/api/latest/interfaces/PropsRange.mdx)
- [`PropsSingle`](/api/latest/interfaces/PropsSingle.mdx)

| `className?` | `string` | The CSS class to add to the container element. |
| `classNames?` | `Partial`\<`Partial`\<[`StyledElement`](/api/latest/type-aliases/StyledElement.mdx)\<`string`\>\>\> | Change the class names used by DayPicker.<br /><br />Use this prop when you need to change the default class names — for example<br />when importing the style via CSS modules or when using a CSS framework. |

| `formatters?` | `Partial`\<[`Formatters`](/api/latest/type-aliases/Formatters.mdx)\> | A map of formatters. Use the formatters to override the default formatting<br />functions. |

| `fromMonth?` | `Date` | The earliest month to start the month navigation. |
| `fromYear?` | `number` | The earliest year to start the month navigation. |
| `hidden?` | [`Matcher`](/api/latest/type-aliases/Matcher.mdx) \| [`Matcher`](/api/latest/type-aliases/Matcher.mdx)[] | Apply the `hidden` modifier to the matching days to hide them from the<br />calendar. |
| ~~`hideHead?`~~ | `boolean` | Hide the month’s head displaying the weekday names.<br /><br />**Deprecated**<br />Use [hideWeekdayRow](/api/latest/interfaces/PropsBase.mdx) instead. |

| `id?` | `string` | A unique id to replace the random generated id – used by DayPicker for<br />accessibility. |
| ~~`initialFocus?`~~ | `boolean` | When a selection mode is set, DayPicker will focus the first selected day<br />(if set) or the today's date (if not disabled).<br /><br />**Deprecated**<br />Use the [autoFocus](/api/latest/interfaces/PropsBase.mdx) prop instead. |
| `labels?` | `Partial`\<[`Labels`](/api/latest/type-aliases/Labels.mdx)\> | Labels creators to override the defaults. Use this prop to customize the<br />ARIA labels attributes. |
| `lang?` | `string` | Add the `lang` attribute to the container element. |

| `modifiers?` | [`DayModifiers`](/api/latest/type-aliases/DayModifiers.mdx) | Map of modifiers to add to the matching days. |
| `modifiersClassNames?` | [`ModifiersClassNames`](/api/latest/type-aliases/ModifiersClassNames.mdx) | Change the class name for the day matching the [modifiers](/api/latest/interfaces/PropsBase.mdx). |

| `month?` | `Date` | The month displayed in the calendar.<br /><br />As opposed to [PropsBase.defaultMonth](/api/latest/interfaces/PropsBase.mdx), use this prop with<br />[PropsBase.onMonthChange](/api/latest/interfaces/PropsBase.mdx) to change the month programmatically. |
