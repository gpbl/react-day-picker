# Interface: RowProps

The props for the [Row](/api/latest/functions/Row.mdx) component.

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `dates` | `Date`[] | The days contained in the week. |
| `displayMonth` | `Date` | The month where the row is displayed. |
| `weekNumber` | `number` | The number of the week to render. |
