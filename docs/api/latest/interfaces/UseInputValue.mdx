# Interface: UseInputValue

Represent the value returned by [useInput](/api/latest/functions/useInput.mdx).

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `dayPickerProps` | [`InputDayPickerProps`](/api/latest/type-aliases/InputDayPickerProps.mdx) | The props to pass to a DayPicker component. |
| `inputProps` | [`InputProps`](/api/latest/type-aliases/InputProps.mdx) | The props to pass to an input field. |
| `reset` | () => `void` | A function to reset to the initial state. |
| `setSelected` | (`day`: `undefined` \| `Date`) => `void` | A function to set the selected day. |
