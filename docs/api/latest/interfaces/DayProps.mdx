# Interface: DayProps

Represent the props used by the [Day](/api/latest/functions/Day.mdx) component.

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `date` | `Date` | The date to render. |
| `displayMonth` | `Date` | The month where the date is displayed. |
