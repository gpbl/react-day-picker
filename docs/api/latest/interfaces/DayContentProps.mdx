# Interface: DayContentProps

Represent the props for the [DayContent](/api/latest/functions/DayContent.mdx) component.

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `activeModifiers` | [`ActiveModifiers`](/api/latest/type-aliases/ActiveModifiers.mdx) | The active modifiers for the given date. |
| `date` | `Date` | The date representing the day. |
| `displayMonth` | `Date` | The month where the day is displayed. |
