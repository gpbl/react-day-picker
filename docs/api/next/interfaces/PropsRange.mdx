# Interface: PropsRange

The props for the range selection mode.

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `defaultSelected?` | [`DateRange`](/api/next/type-aliases/DateRange.mdx) | The initially selected range when not controlled. |
| `max?` | `number` | The maximum number of days that can be selected. |
| `min?` | `number` | The minimum number of days that can be selected. |
| `onSelect?` | [`SelectHandler`](/api/next/type-aliases/SelectHandler.mdx)\<`"range"`\> | The callback called when the user selects a day. |
| `required?` | `boolean` | Makes the selection required. |
| `selected?` | [`DateRange`](/api/next/type-aliases/DateRange.mdx) | The selected range. |
