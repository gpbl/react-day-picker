# Interface: SelectionContext

Share the selected days and the selection methods across the components.

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `excluded` | [`Matcher`](/api/next/type-aliases/Matcher.mdx)[] | The excluded days. |
| `isExcluded` | (`date`: `Date`) => `boolean` | Return `true` if the given day is excluded. |
| `isSelected` | (`date`: `Date`) => `boolean` | Return `true` if the given day is selected. |
| `selected` | [`Selected`](/api/next/type-aliases/Selected.mdx)\<[`Mode`](/api/next/type-aliases/Mode.mdx)\> | The currently selected value. |
| `setSelected` | (`date`: `Date`, `modifiers`: [`Modifiers`](/api/next/type-aliases/Modifiers.mdx), `e`: `MouseEvent`\<`Element`, `MouseEvent`\> \| `KeyboardEvent`\<`Element`\>) => `void` | Set the selected days. |
