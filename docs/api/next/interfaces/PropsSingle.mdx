# Interface: PropsSingle

The props for the single selection mode.

## Properties

| Property | Type | Description |
| :------ | :------ | :------ |
| `defaultSelected?` | `Date` | The initially selected value when not controlled. |
| `onSelect?` | [`SelectHandler`](/docs/api/next/type-aliases/SelectHandler.mdx)\<`"single"`\> | The callback called when the user selects a day. |
| `required?` | `boolean` | Makes the selection required. |
| `selected?` | `Date` | The selected Date. |
