# PropsBase

The base props for the [DayPicker](/docs/api/next/functions/DayPicker.md) component.

These props are used to change the navigation, the styling and the behavior
of the calendar.

## Extended by

- [`DayPickerContext`](/docs/api/next/interfaces/DayPickerContext.md)

### ISOWeek?

> **ISOWeek**?: `boolean`

### autoFocus?

> **autoFocus**?: `boolean`

When a selection mode is set, DayPicker will focus the first selected day
(if set) or the today's date (if not disabled).

Use this prop when you need to focus DayPicker after a user actions, for
improved accessibility.

<a id="captionlayout" name="captionlayout"></a>

### ~~captionLayout?~~

> **captionLayout**?: [`CaptionLayout`](/docs/api/next/type-aliases/CaptionLayout.md)

Change the layout of the caption.

- `buttons`: display prev/right buttons
- `dropdown`: display dropdowns to change the month and the year

**Note:** the `dropdown` layout is available only when `fromDate`,
`fromMonth` or`fromYear` and `toDate`, `toMonth` or `toYear` are set.

#### Deprecated

To show the dropdowns, use [dropdownNavigation](/docs/api/next/interfaces/PropsBase.md#dropdownnavigation). To hide
  the navigation buttons, set [hideNavigation](/docs/api/next/interfaces/PropsBase.md#hidenavigation).

#### Default Value

```ts
buttons
```

[src/types/props.ts:319](https://github.com/gpbl/react-day-picker/blob/next/src/types/props.ts#L319)

<a id="dropdownnavigation" name="dropdownnavigation"></a>

### dropdownNavigation?

> **dropdownNavigation**?: `boolean` \| `"month"` \| `"year"`

Show dropdowns to navigate between months or years.

- `true`: display the dropdowns for both month and year.
- `month`: display the dropdown for the month.
- `year`: display the dropdown for the year.
- `false`: hide the dropdowns.

**Note:** showing the dropdown will default [fromYear](/docs/api/next/interfaces/PropsBase.md#fromyear) to the 100
years ago, and [toYear](/docs/api/next/interfaces/PropsBase.md#toyear) to the current year.

#### Source
