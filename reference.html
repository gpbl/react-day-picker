Summary

Looking for the props for DayPicker? See:

  • DayPickerBase - The base interface for DayPicker props, extended by:
    • DayPickerDefaultProps - when DayPicker is in the default or custom selection mode.
    • DayPickerSingleProps - when DayPicker is in single selection mode.
    • DayPickerMultipleProps - when DayPicker is in multiple selection mode.
    • DayPickerRangeProps - when DayPicker is in range selection mode.
